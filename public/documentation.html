Documentation

This is an API that will give users access to various data about vinyl records including (but not limited to) genres, artists, and release dates. With this database, users can register with their own personal information and add albums that they already own and albums on their wish list.

Album Endpoints

Business Logic URL HTTP Method Request format Response format
Get the list of all albums /Albums GET None JSON object holding data about all albums
Get the data about a single movie by title /Albums/[Title] GET None JSON object holding data about single movie containing a title,description,genre,director,image URL.Example:
{
_id : ObjectId("5f3c992a2e8816c77fd833b2"),
Title : "All Eyez On Me",
Genre: "Rap",
Artist: "2Pac",
ImagePath: "alleyezonme.png" ,
}
Get all albums by genre /Albums/Genre GET None Rap

User Requests

Business Logic URL HTTP Method Request format Response format
Create new user /users/ POST A JSON object holding data about user to add with following structure:
{
Name: "User One",
Password:"userone",
Email: "userone@gmail.com",
Birthday:"2001-01-01",
OwnList: [],
WantList: []
}
 JSON object holding data about user account that was just created. Example:
{
Name: "User One",
Password:"userone",
Email: "userone@gmail.com",
Birthday:"2001-01-01",
OwnList: [],
WantList: []
}
Get/user profile /users/[Username] GET JSON object holding data about user account with the following structure:
{
Name: "User One",
Password:"userone",
Email: "userone@gmail.com",
Birthday:"2001-01-01",
OwnList: [],
WantList: []
}
Update user information (username,,name, password, email, date of birth) /users/[Username] PUT A JSON object holding data about user to be updated with following structure:
{
Name: "User One",
Password:"userone",
Email: "userone@gmail.com",
Birthday:"2001-01-01",
OwnList: [],
WantList: []
} 		A text message indicating whether the user info was successfully updated
Add an album to user's list of albums they own /users/[Username]/Albums/[AlbumID] POST A JSON object holding data about album ID to be added to user's list of favourites.Example:
{
AlbumID:"_id" : ObjectId("5df2d3bcf0c59ce6e6d51b89"
}
 JSON object holding data about updated user profile including the updated list of owned albums.Example:
{
Name: "User Two",
Password:"usertwo",
Email: "usertwo@gmail.com",
Birthday:"2002-02-02",
OwnList: [ObjectId("5f3c99b42e8816c77fd833b3"), ObjectId("5f3c9a7b2e8816c77fd833b4"), ObjectId("5f3c9ad62e8816c77fd833b5")]
}
Remove a movie from user's list of owned albums /users/[Username]/Albums/[AlbumID] DELETE A JSON object holding data about MovieID to be removed from user's list of owed albums.Example:
{
AlbumID:"_id" : ObjectId("5f3c9ad62e8816c77fd833b5")
}
 JSON object holding data about updated user profile including the updated list of owned albums.Example:
{
Name: "User Two",
Password:"usertwo",
Email: "usertwo@gmail.com",
Birthday:"2002-02-02",
OwnList: [ObjectId("5f3c99b42e8816c77fd833b3"), ObjectId("5f3c9a7b2e8816c77fd833b4")]
}
Add an album to user's list of albums they want /users/[Username]/Albums/[AlbumID] POST A JSON object holding data about album ID to be added to user's list of favourites.Example:
{
AlbumID:"_id" : ObjectId("5f3f3fe82e8816c77fd8342b"
}
 JSON object holding data about updated user profile including the updated list of wanted albums.Example:
{
Name: "User Two",

Birthday:"2002-02-02",
OwnList: [ObjectId("5f3f3fe82e8816c77fd8342b"), ObjectId("5f3f40192e8816c77fd8342c"), ObjectId("5f3f403f2e8816c77fd8342d")]
}
Remove an album from user's list of wanted albums /users/[Username]/Albums/[AlbumID] DELETE A JSON object holding data about AlbumID to be removed from user's list of wanted albums.Example:
{
AlbumID:"_id" : ObjectId("5f3f3fe82e8816c77fd8342b")
}

Name: "User Two",
